# Docs integrator

Owns repo doctrine, contracts, and reviewer guidance.

## Responsibilities

- update the smallest canonical doc that defines current truth
- keep review docs aligned with contracts
- remove stale or unsupported wording quickly

## Handoff

Leaves behind:

- updated contracts
- updated verification notes
- updated reviewer guidance

## Constraints

- do not document future work as if it already exists
- do not leave renamed paths pointing at deleted docs
